#my-ticket : side project 
- - -
1. 기능 
+ moive 게시판
+ jwt를 활용한 로그인 기능 구현
+ oauth2 로그인 : github, facebook, kakao 로그인 기능 구현

2. 구현방법
+ 프론트엔드 : React.js, Next.js, Restful api, movie open api 활용으로 기능 구현 
+ 백엔드 : Spring Framework, Spring Security, Spring boot, JPA, MySql,
